# Metrics demo

Demo for meetup "How to add metrics in frontend project".

Technologies:

- React (NextJs),
- [prom-client](https://github.com/siimon/prom-client)
- Express
- Axios


# Setup and run

Setup:

```
yarn install
```

Run in dev mode: 

```
yarn dev
```

Run in production mode:

```
yarn build
yarn start
```

Application will be available at: [http://localhost:3000](http://localhost:3000).
